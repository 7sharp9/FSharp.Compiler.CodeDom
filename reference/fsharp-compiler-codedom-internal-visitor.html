Signature: f:(('?17182 -> '?17183 -> '?17184 -> '?17183) -> '?17182 -> '?17183 -> obj -> '?17183) -> scope:'?17182 -> st:'?17183 -> o:'?17185 -> '?17183
Type parameters: '?17182, '?17183, '?17184, '?17185

Walks through the CodeDom tree and keeps current "scope" and the result. The result is collected through entire tree, but the modified scope is passed only to sub-nodes of the current node.

First argument is a function that is called for nodes and has a function as a first argument, scope and result as a second and current node as a third. The function argument can be used to walk deeper in the tree if wanted.
